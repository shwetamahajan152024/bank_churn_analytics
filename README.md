##Project Overview
Analyzed 10000 bank customer recprds to identify churn patterns using Python and SQL.

##Tools Used
- Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
- SQLite + SQLAlchemy
- Google Colab

##Key Insights
- Overall churn rate: 20.38%
- Germany had highest churn: 32.44%
- Customers above 60 churned at 56.21%
- Inactive members churned at 2x the rate of active members
- Highest risk segment: Inactive Germen customer aged -58

##Project Structure
- Phase 1: Exploratory Data Analysis
- Phase 2: Data Cleaning and Label Encoding
- Phase 3: SQLite Database Integration
- Phase 4: SQL Business Insights queries
